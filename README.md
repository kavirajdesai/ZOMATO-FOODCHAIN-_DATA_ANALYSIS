#Zomato Data Analysis

Overview:
An exploratory data analysis (EDA) project on restaurant and food delivery data to uncover trends in customer preferences and restaurant performance.

Tools & Technologies: Python (Pandas, NumPy, Seaborn, Matplotlib), SQL

Key Highlights:

Cleaned and analyzed large restaurant datasets using Python and SQL.

Created visualizations showing location-wise ratings, pricing patterns, and cuisine popularity.

Engineered regression-ready features that boosted model accuracy to 88% in predictive analysis.
